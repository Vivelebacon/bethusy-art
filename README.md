# Béthusy-Art & Design — Refonte (proposition)

Redesign demo of https://www.bethusy-art.ch/ by Hugo Web Design.

Single-file site: `index.html` (GSAP + ScrollTrigger + Lenis).
Hero motion (`assets/hero-loop.mp4`) is a Ken Burns montage built from the studio's own
published interior photos; gallery/service images are served live from bethusy-art.ch.

## Deploy
Static. From this folder: `vercel --prod` (or import the repo on Vercel).
